# KotlinProject
## Проект **FitHelper**

Данный проект написан на языке Kotlin и позволяет создавать, сохранять и просматривать комплекс физических упражнений для занятий в фитнесцентре.
В приложении реализована возможность хранения компекса упражнений в виже записей базы данных, доступ к которым очуществляется через Google аккаунт пользователя приложением.
